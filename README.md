**US States Salary & Profit Analysis – Statistical Correlation Study**

## Project Description
This project explores profit metrics across 50 US states, focusing on average profit, total profit, population ranking, and profit ranking. The analysis includes data cleaning, median imputation for zero values, and statistical relationship analysis between population size and profit levels.

## Key Objectives
- Clean and prepare US states salary/profit dataset
- Analyze the relationship between population ranking and profit
- Identify patterns and insights in state-level economic data
- Test the hypothesis that larger population states have lower average profit

## What I Did
- Performed data cleaning and handled zero values using median imputation
- Conducted Exploratory Data Analysis (EDA) on profit distribution
- Applied Spearman correlation to examine the relationship between population ranking and profit ranking
- Created visualizations to show distributions and trends

## Key Findings
- No meaningful correlation was found between population ranking and profit ranking (Spearman correlation = 0.0035, p-value = 0.9806)
- State population size alone is not a strong predictor of average profit
- Other factors such as industry composition, education levels, and economic structure likely play more significant roles

## Technical Skills Demonstrated
- Data cleaning and feature engineering (pandas)
- Statistical analysis (Spearman correlation)
- Data visualization (Seaborn & Matplotlib)
- Hypothesis testing and interpretation

## Tools Used
- Python (pandas, numpy, scipy, seaborn, matplotlib)
- Jupyter Notebook

## Business Insights & Recommendations
- Companies and policymakers should look beyond population size when analyzing economic performance
- Focus on industry specialization, education, and infrastructure rather than just state size**US States Salary & Profit Analysis – Statistical Correlation Study**
